

## About

This is my new reworked high tech wasm homepage!
Based on [`yew-wasm-pack-template`](https://github.com/yewstack/yew-wasm-pack-template)

## 🚴 Usage

```
yarn install
```

### 🛠️ Build with `yarn run build`

```
yarn run build
```

### 🔬 Serve locally with `yarn run start:dev`

```
yarn run start:dev
```
